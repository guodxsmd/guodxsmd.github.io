致敬清清风大佬

目前渣胖二服开的BT版，经验获取难度高，消耗力度大，瞎鸡儿搞。清风大佬的这篇老服攻略可能有不太准确的地方，不过总体来讲出入并不大，有些地方后期慢慢修改吧。

2、土豪先天： 30.30.11.13

二、初入江湖

  1. 刚注册的新号会出生在北帝庙，找红龄打命令/ask baohu，要3个保命金丹。
  2. 北帝庙找新手任务员领一个侍卫，然后做新手任务–初入江湖：去兵器房拿书和装备，把书用了，装备装上。然后去练功房，把木人杀了，可以得到1500经验和2500点内力上限。(主要是为了2500的内力)
  3. 打鸟，脚本在客户端压缩包里自带，不能运行请到群共享查询下载。大概3个小时可以挂500W经验。
  4. 打鸟无经验后，升级到90，到扬州西北武庙，拜新手导师：一般有一个少林导师和灵鹫导师，无拜师选项右键点击导师会出菜单。
  5. 按以下顺序请教主要技能到311级：
    少林为：读书写字（401级）、满基本剑法，满无量剑法，满基本招架，100级无量心法，100级无量神功，满基本躲闪。剩下如果有经验就升级。准备技能：基本剑法==>无量剑法、基本招架==>无量剑法、基本内功==>无量神功。 在116级之后补上400级基本内和北冥神功，如果愿意也可以学400级基本棍和中平枪（据说可以加命中）
    灵鹫为：读书写字（401级）、基本招架、基本指法、折梅手、柔情指、八荒六合唯我独尊功、基本躲闪。准备技能：基本指法==>柔情指、基本招架==>折梅手、基本内功==>八荒。折梅绝招梅花三弄需要解密，解密条件是找个男号结婚并离婚（结婚地点：!go 杭州红娘庄），去之前找老玩家要个订房服务，吃掉后变16岁才可以结婚。
    其他技能都不是必需品【无需请教】
  6. 到扬州武庙等待，下跪，并大喊：新人报道已打完鸟求新手装备一套（无人理会的话，请过段时间继续练习乞讨术。）
  7. 新手装备：倚天剑/倚地剑（攻击+1000/+500，重启不消失，死亡有保命不掉，没保命掉落），匕首（副手攻击+200，可存柜子），青布面纱(免伤5000点，不用穿其他衣服了，需90级才可装备)。回血（金）丹/回内（金）丹，储存大量HP和MP，战斗开始即补满血内，可持续作战提升效率。
  8. 戴好青布面纱（免伤5000，泰山怪一般打不动你了；无装备选项用/wield命令），拿好倚*剑和匕首，吃掉回血回内丹，就可以开始江湖(挂机/脚本/VPS)之路了。
  9. 技能和装备准备好之后，去刷心魔，然后刷泰山副本，每次完成泰山副本之后都升级（转生前等级越高，泰山副本的经验越多），从90级升级到116级之后去把之前的技能补上来，到此技能全部补齐。先装备上北冥神功去泰山把内力吸到6000+（有千年寒玉可以到8000+，有脚本），之后继续刷泰山有经验就继续升级，直到转生。

新服这里打完鸟之后可以直接送菜、做北京任务直接到转，转了接着北京或者搞去拉车，泰山跟襄阳这里暂时是废的。

三、经验获取及日常

  1. 【心魔】，一天一次，12点刷新。给当前级别升下一级所需经验（50W封顶）和5点内力上限，对所有副本经验有10次10倍加层！可累计次数，该次数重启不消失。每日必做 —— 新号戴面纱可随便打。
  2. 泰山副本。找泰山山脚的老樵夫，谈话进入，杀20个NPC为一轮。新号戴面纱可随便打。新号主要经验来源。每轮1-3万经验，享受心魔的10倍加层。
  3. 襄阳副本。襄阳将军府找郭靖谈话保卫襄阳，一路杀进去干死最后BOSS蒙哥。116级满技能新号戴面纱可随便打。新号主要经验来源。每轮1-3万经验，享受心魔的10倍加层。偶尔掉神秘宝箱。吃回内丹才可进入，BOSS会有连续5战。
    不吃回内的刷法：1、150级以上穿自制铠甲，进本前睡觉补满内力，无伤刷完；2、先天敏捷超过21的号戴回龙壁和剑盾珠索；3、云龙弟子戴青龙鳞；4、穿蟾蜍皮衣；5、戴白虎追风护腕。
  4. 北京任务。即原北京任务，找北京张府的张居正索要任务，杀死指定NPC，回复任务。新号戴面纱可随便打。新号主要挂机经验来源，基本无消耗无危险，不享受心魔加层，效率100左右。完成任务的奖励点数可兑换寻宝，效率很低。
  5. 日常任务。分别为：白马啸西风，笑傲江湖，连城诀，碧血剑，神雕侠侣，飞狐外传这六本书的任务（倚天屠龙记需要小号配合，无人做）。116级满技能新号戴面纱可随便打。建议每日一轮，刷新时间为上次完成时间。年龄越高经验越多，每轮有几十万经验和几十万声望。
  6. 恶人谷副本。到西域草原观看路牌可进入。最为三个入口分别为经验奖励，内力上限奖励及技能奖励。116级满技能新号戴面纱可随便打。每次经验奖励为50W。
  7. 保卫萧峰。–2服除丐帮外该副本无人问津，后面的说明不是丐帮号不用看了——先找郭靖谈话刷一次襄阳副本，然后再找郭靖谈话问降龙十八掌，再到扬州密林10找洪七公谈话问降龙十八掌，再去灵鹫独尊厅找竹虚谈话问降龙十八掌，再到江夏飞龙阁找金庸大师进宋辽战场，一路杀到临潢府。杀辽军将军有几率得腰牌。得到腰牌后在临潢府观察本地，进入皇宫别院，找萧峰谈话任务即可开始。经验跟萧峰的状态有关，状态越高经验越高。
    清风推荐经验路线：每日心魔和日常找个固定时间一起做，然后挂北京任务。等通天、节日、当机等情况开多倍（一般是2倍、4倍、5倍等）之后，集中挂泰山和襄阳（这样刷一次就是50倍经验，一轮泰山100万经验爽的飞起）。 目前等渣胖开多倍经验和心魔加成吧

四、转前准备

  1. 如果内力没有吸满，学满北冥，去普通泰山（即老泰山，非副本）将内力吸到6000+；如果找人借个千年寒玉可吸内力上限至8000+，大约半小时即可搞定（借个缀玉华袍穿上只需10分钟）。
  2. 主要技能学到400级满。
  3. 找老玩家借1000W金币。
  4. 记得打经验。
  5. 【150级之后记得到群里乞讨一套自制（铠甲+副手）】

五、转生

  1. 新号升级到165级，身上存有1200W经验时就可以转生了。死亡经验和请教经验也计算在总经验里，所以转生前可以放心死。
  2. 转生后各先天属性增加10%，血量翻两倍多一点，升级经验提升5倍，请教经验增加20%。所以本服先天体质需要适当考虑。
  3. 转生人物的门派、技能、声望、师傅等全部清零，最多可以保留1518W经验。部分门派开放转生技能。
  4. 转生流程：扬州南门–江夏城–飞龙阁–金庸大师–谈话（查询转生后的保留经验，不到1518万请继续回去杀经验）–谈话（我要转生）–给金庸大师1000W金币–我要转生。切记：先谈话在给钱！！！
  5. 广告：转生重要道具《转生技能保留》钱庄大胖子2000点卷购买，转生可以保留全部技能，转后经验可以全部用来升级，刷经验超快~(土豪必备，屌丝就算了，不差那几百万经验)

目前获取经验最快的方式：开寻宝，刷天下会帖子，去狂肝天下会，刷荣誉买经验神丹，一颗1000W经验。

六、转生后

转生后，升到79级，最多到80级不要再升了，不然没经验请教。
继续拜武庙导师，学无量技能。

七、转生后的经验来源

  1. 心魔、日常、泰山、襄阳、北京、恶人谷，同转前一样。
  2. 保镖，较复杂不建议新手选择：到福州福威镖局找林震西领任务，18W为巨车。需要等级转生130级以上，3个号。推荐配置1：明教 神龙 小号。其中明教缠、神龙伤，小号跟着混。配置2：明教缠、古墓伤、峨嵋回内
  3. 金钱帮分舵副本，转生116技能满，穿转生衣服即可挑战。会员每日礼包里有一张门票,进入杀光里面所有NPC即可。50W金币和经验，有几率掉落金条，金条使用会获得1000W金币。
  4. 天下会副本，转生116级技能满，穿桃/红花转套，有缠有连，吃朱果即可尝试挑战，150级以上穿自制铠比较安全。需要寻宝出的天下会挑战贴，任何地点使用即可进本。4个BOSS，杀光可得6万经验，少量掉落独孤九剑残篇和天下会秘籍。享受心魔10倍加层和服务器加倍。攒点帖子到周五周六集中使用，10倍心魔+服务器4倍加层，每次240万+经验，很可观。

八、转生后内力上限

  1. 400基本内+400高内可打坐到12000；400基本内+400高内+千年寒玉可打坐到16000；推荐带寒玉床，每次打坐可加10点上限，极大提升效率（8000-16000一天即可完成）。
  2. 16000之后吃人参（+3）和人参果（+10）到18000。
  3. 吃腊八粥（+100）到20000.
    4、丹枫，一次性吃大量丹枫（少于1000个没必要吃……）然后打坐。
  4. 百香花，加内力上限5点，无上限，开神秘宝箱或寻宝可以获得
  5. 其实吧，这个吧，不好吧，还是说了吧：转后其实可以500高内……方法一：做天龙寺任务，选择打晕段玉给鸠摩智的话，每次奖励1级小无相，500封顶。方法二：做逍遥秘籍任务，每次重启后去无量山玉璧磕头，出的逍遥秘籍有一种可以读北冥到500.
  6. 内力上限：转前为：maxmp=有效内功等级10；转后为：maxmp=有效内功等级20。有效内功等级的计算方式为：高内等级+基本内功/2

九、飞升及轮回

  1. 转生后达到165级即可飞升，无需存经验。
  2. 找江夏飞龙阁大胖子谈话，选飞升试练，杀死大胖子，该副本可重复挑战，不会死亡。大胖子100万血，有大约2万免伤。进战斗立即缠，缠中后刷几次技能后及时补缠，确保胖子一直自顾不暇，一直缠到死完事。伤害不足的吃攻击神丹或者借个逐日耳环，内力不够的带几个无常丹战斗中使用。
  3. 然后重新与大胖子谈话，选飞升，给大胖子1亿金币，点我要飞升即可。
  4. 飞升后清空门派声望和师傅，保留全部技能，血量变为初始值的5倍略少一点，攻防增加1/3，取消技能上限的限制，所有门派开放飞升专属技能（需正确穿戴全套门派飞升套装）。
  5. 飞升后送1.5亿经验，直接升级到160即可，剩1500万经验请教足够。
  6. 飞升165级之后，可以用3亿经验+轮回再生丸到江夏胖子处轮回，每次+1属性点，可自由分配，算先天属性。每轮回一次，轮回再生丸需求数量+1.
  7. 【因为技能总数不影响练习速度和读书速度，只影响请教所需经验】飞升前建议到武庙导师补完全部基本和感觉有用的技能。

十、套装及圣衣

  1. 24个门派均有专属门派套装，另有小宝套、金箍棒等特殊套装
  2. 飞升套装只能本门派穿戴，普通套装和转生套装无门派限制，只要达到套装各部位的装备要求即可穿戴，但是不能混穿（即使同门派的普通、转、飞套也不能混穿）
  3. 套装按一定顺序穿戴，可激活特殊效果，激活表现为一行或两行红字，无红字即未激活。
  4. 普通套装不能合成。转生套装+转生合成符可合成转生圣衣（例如：青布面纱·转+碧玉剑萧·转+奇门法袍·转+桃花转生合成符 = 桃花圣衣·转）
  5. 飞升套装+飞升合成符+转生圣衣 可合成飞升圣衣（例如：青布面纱·飞+碧玉剑萧·飞+奇门法袍·飞+桃花飞升合成符 + 桃花圣衣·转 = 桃花圣衣·飞）
  6. 圣衣合成成功率一般在10%左右，失败会随机只剩一件套装散件，圣衣和其他散件消失。
  7. 圣衣的优势：穿戴方便，1件顶过去5件，一口气上五楼，不费劲！部分门派套装散件太多，穿齐后无法拿主手或副手或项链或戒指，合成圣衣后不占装备位置。装备即激活特殊效果，无顺序限制。不会被打掉武器导致套装效果消失。
  8. 合成圣衣后，个别门派武器的部分特殊效果会消失（例如武当的声望克制等。另外：红花白龙剑的定身、星宿神木王鼎的吸血补内、古墓的重剑加成等已修复）。
  9. 转生套装的特殊属性：转生套装各散件的护甲，攻击，寻宝几率翻倍，衣服散件增加100000HP，其他属性不变。
  10. 飞升套装的特殊属性：飞升套装武器增加20000伤害（套装里没武器的会加在某一个散件上），衣服增加50000免伤，穿出红字后可以使用门派飞升技能。
  11. 、转生和飞升套装的武器都可以装备左手，攻击力变为1/3。

获取方式：

  1. 寻宝：可出绝大部分普通套装及转生套装散件，部分转生散件几率很低或直接就寻不出（例如青布面纱·转，玄武道袍·转）。
  2. 神秘宝箱：可出少量普通套装散件。（寻宝出，襄阳低概率出，杀有经验的NPC低概率出）
  3. 黄金礼包：可出所有转生套装散件。（寻宝出）
  4. 紫金宝盒：可出所有转生套装散件、飞升套装散件、转生和飞升合成符。（寻宝少量掉落，通天塔掉落大量）
  5. 铂金宝盒：可出所有转生套装散件、飞升套装散件、转生和飞升合成符。（只有通天塔出）
